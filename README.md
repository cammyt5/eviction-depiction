# The LLC Next Door: An analysis of corporate housing ownership (by corporation type) impacting eviction rates in Boston

Developed for MIT class 11.C35 in Spring 2024 by Cameron Dougal, Brian Zheng, Aleksandar Jovanovic-Hacon, and Max Katz-Christy.

https://cammyt5.github.io/eviction-depiction/

## Developing

Install dependencies with `npm install` and start a development server:

```bash
npm run dev -- --open
```
